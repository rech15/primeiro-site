Table

A tag que foi utilizada para criar uma tabela HTML, dentro dela, incluí­mos todos os elementos que compõem nossa tabela, ou seja, as células da tabela. Assim sendo, os elementos de uma tabela consistem em outras tags que poderão ser utilizadas.

Del

O elemento HTML representa uma parte do texto que foi excluída de um documento. Este elemento é renderizado, mas não é necessário, pelos navegadores com uma linha entre o texto.

Ins

Tem a utilidade de marcar o texto que foi inserido em um determinado conteúdo desde sua versão anterior, o novo conteúdo é exibido sublinhado, podendo alterar isso quanto quiser com folhas de estilo.

Sub

O marcador sub é utilizado para escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo.

Sup

O marcador sup(Superscript Element) é utilizado escrever um texto com letras menores e com o alinhamento acima das que do resto do conteúdo.

Blockquote

O elemento HTML indica que o texto incluí­do é uma longa citação. A URL para a fonte da citação pode ser dada usando o atributo cite, enquanto uma representação de texto da fonte pode ser dada usando o cite elemento.

Q

Esse elemento HTML aponta que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas; para citações maiores use o elemento blockquote.

Abbr

O Elemento HTML representa uma abreviação. Se presente, o atributo title deve conter a descrição completa, apenas ela.

Address

A tag define as informações de contato do autor ou do proprietário de um documento/artigo. Se o elemento address está dentro do elemento body, ele acaba fugindo das informações acima e representa informações de contato para o documento.

Cite

Esse elemento HTML representa uma referência a um trabalho normalmente artístico, incluindo o título do trabalho ou uma URL de referência, que pode ser em uma forma abreviada.

Bdo

O elemento HTML é utilizado para substituir a direcionalidade atual do texto.

Code

O elemento HTML apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código.

Pre

A tag é utilizada para representar texto, o texto dentro desse elemento é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo. Espaços em branco sÃ£o mantidos no texto da mesma forma em que este foi digitado.

Kbd

Define a entrada do teclado.
